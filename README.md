# connstate-collectd-plugin
Collectd linux network connections states tracking plugin.

#### Metrics
```
ESTABLISHED
SYN_SENT
SYN_RECV
FIN_WAIT1
FIN_WAIT2
TIME_WAIT
CLOSE
CLOSE_WAIT
LAST_ACK
LISTEN
CLOSING
```

#### Protocols
```
TCP
UDP
TCP6
UDP6
```
